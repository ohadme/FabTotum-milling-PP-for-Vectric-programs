Those files are Post FabTotum milling post processors for Vectric programs

Start any Vectric program (Aspire, Cut2D, Vcarve)
On the top menu go to File > Open Application Data Folder.
This will open a new window showing all the program's support folders.
Select the Postp folder and save the new FabTotum *.pp files. [FabTotum_NC_Inch.pp, FabTotum_NC_mm.pp]
